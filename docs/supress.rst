大象实验：


-----------------------------------------------------------------------------------------------------

在马戏团里，有一头成年大象被一根细细的绳子拴在小木桩上，安静地站着，从不挣扎。游客们常常好奇：以大象的力量，明明可以轻易拔起木桩，为什么它从不尝试逃脱？

驯象师解释道：当这头大象还是小象时，它就被同样的绳子和木桩拴住。那时它拼命挣扎，却因力气太小无法挣脱。日复一日，小象渐渐相信——“这根绳子是无法挣脱的”。即使它长大后力量增长了百倍，也再不会尝试挣脱，因为“不可能”的信念早已深深刻在它的心里。

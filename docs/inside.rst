随笔三十六 向内求
======================

心外无物，心外无物，心外无物。

你就是世界（一切美好，都是你大脑处理后的美，一切讨厌，同样都是你大脑处理后的厌恶，世界本身没有任何剧烈变化，变化的是你的心）

向外求全是埋怨，向内求全是答案。

带着觉知，理解自己，不再做情绪的奴隶，笑傲江湖。

人的疼痛苦，除了疾病导致的，其余的都是自己的心(除了肉体的痛是真实疼痛，其它疼痛都是你自己大脑给你放的电影，虚幻、虚幻、虚幻)。

顺其自然，允许一切发生，一切发生皆有利于我。

​人生哪能多如意，​万事只求半称心。
